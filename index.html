Bootstrap Column Syntax

Week 2: July 20

javascript

bootstrap

As mentioned before, bootstrap has 12 columns. To use those columns you need to use the col-md-# format. This stands for column-screensize-number of columns. The screen sizes are represented with xs, sm, md, and lg. As shown below, these different screen sizes are meant to represent the different types of devices. xs is for phones, sm is for tablets, md (which I use in my examples) represents laptop computers and lg is for desktop monitors. When calling the column styling, you need to remember that these are all lowercase. Bootstrap will not understand the class if you use camel case or capital letters.

Colors

Week 1

css

When choosing colors for a company or for on a website, you have to have an idea of how to call specific colors with your CSS code. Most programmers will use the hexadecimal layout. This contains 6 spots that follow a pound sign (#). The first two, as shown below, represent the color red, the second pair represents green and the third, blue. with the hexadecimal layout, you can use 0-9 and a-f. 0 means that that spot has none of the color, and with f, it has the most it can have of that color. For example: #000000 is Black and #ffffff is white. #ff0000 is red, #00ff00 is the greenest of greens and #0000ff is the bluest of blues.

CSS also supports RGBA colors, HSL colors, and HSLA colors, which you can find more information about at W3Schools.com.

Declaring a Variable

Week 1

javascript

Variables are very important in JavaScript. The definition of a variable according to W3Schools.com is

"Variables are containers for storing data values."

The keyword used to declare variables is var. This is a very imporatant part of javascript. When you declare a variable, you need to use the keyword "var" and also give the variable a name. The name can be anything you want but there are some important rules. Mainly, the best way to write a variable name is to use camelCase and try not to use numbers at the beginning or end of the name.camelCase is using a lowercase letter for the first word in the variable name and an uppercase letter for any following word. Do not use spaces in variable names!

one thing to be aware of is that you can declare a variable and not have a value attached to it right away. More into that later, though.

HTML and CSS

July 4, 2015

html

css

What is HTML?
HTML stands for Hyper Text Markup Language. HTML is the structural outline of a web page. Without CSS, an HTML page would just be words. No style, no font changes, no color at all. HTML is, however, very important to have. It includes the Head of the page which is needed to link the CSS to it and the Body of the page which includes the main content of the page. Within the Body can be the Header, Main, and Footer sections to separate the different parts of the web page and stay more organized.

What is CSS?
CSS stands for Cascade Styling Sheets. CSS is the Style of the page. This includes: font styles, sizes and colors, and background colors and images. Different sections (like the header, main, and footer sections mentioned above) can, if you so choose, have different styles each. CSS is the overall appearance of the webpage. You can use CSS to control the style and layout of multiple Web pages all at once.

According to Wikipedia, Software Development is the computer programming, documenting, testing, and bug fixing involved in creating and maintaining applications and frameworks involved in a software release life cycle and resulting in a software product. The term refers to a process of writing and maintaining the source code, but in a broader sense of the term it includes all that is involved between the conception of the desired software through to the final manifestation of the software, ideally in a planned and structured process.
